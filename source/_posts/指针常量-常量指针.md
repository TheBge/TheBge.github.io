pointer to const 指向常量的指针  
const pointer 常量指针  
```c
int *p =&x;//p为指针,指针指向int类型 x     
const int *p = &x; //指针p,指针指向const int类型的x       
int *const p = &x; //指针是const p 地址为常量，说明指针指向的对象不能改变      
const int *const p =&x; //指针是const p，指向cosnt int类型的x
```
